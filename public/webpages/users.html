Users API Endpoint

About

This endpoint perform operations on the users stored in the MongoDB database. You have to send your requests to /api/users. It responds to 2 HTTP methods, GET and POST.

Visit Code

GET  /api/users

A GET request to this endpoint, returns a list of all available users stored in the MongoDB database.

GET for user info /api/users/me

A GET request to this endpoint with x-auth-token in the request header, returns all the information about the user that belongs to the x-auth-token, and to get the value of x-auth-token visit Auth endpoint.

POST   /api/users

We can post a new user via POST request, by including the following data in our request body.

 "name": "testUser1",
 "email": "testUser1@gmail.com",
 "password": "testUser1Password"

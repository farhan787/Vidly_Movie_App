Rentals API Endpoint

About

This endpoint perform operations on the rentals stored in the MongoDB database. You have to send your requests to /api/rentals. It responds to 2 HTTP methods, GET and POST.

Visit Code

GET  /api/rentals

A GET request to this endpoint, returns a list of all the rentals stored in the MongoDB database.

POST  /api/rentals

A customer can get a movie on rent by sending a POST request to this endpoint with customer id and movie id in the body of the request and also send the x-auth-token in the request header for authentication, x-auth-token will be the authentication token, that we get from the POST method of Auth endpoint..

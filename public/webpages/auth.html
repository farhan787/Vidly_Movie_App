Auth API Endpoint

About

This endpoint authenticates the users stored in the database. It returns JSON Web Token (JWT) which contains all the information about that user and will help users to perform many operations in other endpoints as well. Send your request to /api/auth. It responds to only 1 POST HTTP method.

POST  /api/auth

Send a POST request to this endpoint using below data in your reqeust body. It will return a JSON Web Token (JWT) which you can use as x-auth-token in the header of the request while using various POST requests for different endpoints, so the server will be able to identify you, whether you are a trusted user or some malicious client who wants to access our database.

Returns API Endpoint

About

This endpoint returns the movie that a customer has taken on rent and calculate the rental fee and other stuff related to the movie rent. Send request to /api/returns. It responds to only 1 POST HTTP method.

Visit Code

POST   /api/returns

Send a POST request to this endpoint. Send the customer id and movie id in the body of the request, also send x-auth-token in the request header and the value for this x-auth-token will be the authentication token, that we get from the POST method of Auth endpoint.

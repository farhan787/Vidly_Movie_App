Vidly Movie App

About

Vidly Movie App is a backend api app which implements the CRUD operations, authentication, authorisation and various backend stuff using server side technologies. This backend app can be interpreted as a backend prototype of Netflix and other movie streaming services. A user can make his account, can be authenticated, can be registered as a customer and see the available genres (category) of movies, can also see the available movies and can take a movie on rent and can return the movie whenever he/she wants to return and has to pay the rental fee.
